#  Graph Matching Framework

This repository provides an implementation of a graph matching framework based on node embeddings.

## Installation

#### Prerequisites
- CMake >= v.3.10
- C++ Compiler >= C++17

#### Install
```bash
mkdir build
cd build
cmake -S .. -B .
make
```

## How to use

#### Run
```bash
./graph-matching <graph_folder> <node_embeddings_file> <output_path>
```
